# 🎧 AI Audiobook Reader
A Python-based application that converts text from PDF documents into spoken audio. This project uses Text-to-Speech (TTS) technology to make reading more accessible,
using the pyttsx3 and PyPDF2 libraries in Python

## ✨ Features
PDF Text Extraction: Automatically reads content from any PDF file.
Graphical User Interface: Includes a file explorer for easy document selection.
Offline Functionality: Converts text to speech without requiring an internet connection.
Modular Code: Structured with a clear separation between the entry point and game logic.
